Java User Group Bangladesh (JUGBD)

About The Conference

In Bangladesh, the Java Developers' Conference is one of the most successful community-driven software conferences, and it's organized by the Bangladesh Java User Group. The conference helps developers to learn the cutting edge technologies about Java, Web, Mobile, Big Data, Cloud, DevOps, Agile and many more. This also gives the opportunity to developers and software industries to establish networks among them.
